# Matter.js Attraction

```bash
uv run python simulation/matter_js_attraction/matter_js_attraction.py
```

- 100 movers orbit around a central attractor body using gravitational force.
- Each mover is attracted with force proportional to 1/r² (with distance constrained to [5, 25]).
- Press `s` to save a screenshot to `simulation/matter_js_attraction/screenshots/`.

# Simple Spring With Toxiclibs

```bash
uv run python physics/simple_spring_with_toxiclibs/simple_spring_with_toxiclibs.py
```

- Two Verlet particles are connected by a spring.
- The first particle is fixed; drag with mouse to move the second one.
- Press `s` to save a screenshot to `physics/simple_spring_with_toxiclibs/screenshots/`.

# Crowd Path Following

```bash
uv run python simulation/crowd_path_following/crowd_path_following.py
```

- 120 vehicles follow a closed path while separating from each other.
- Press `d` to toggle debug visualization (predicted position, normal point, target).
- Click to add a new vehicle.
- Press `s` to save a screenshot to `simulation/crowd_path_following/screenshots/`.

# Branch Thickness

```bash
uv run python simulation/branch_thickness/branch_thickness.py
```

- Recursive tree visualization with branch thickness based on length.
- Move the mouse left and right to change the branching angle (0 to 90 degrees).
- Each branch is drawn with a stroke weight proportional to its length.
- Press `s` to save a screenshot to `simulation/branch_thickness/screenshots/`.

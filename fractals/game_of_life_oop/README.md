# Game Of Life OOP

```bash
uv run python simulation/game_of_life_oop/game_of_life_oop.py
```

- Object-oriented Conway's Game of Life with `Cell` objects.
- Cell colors: born = blue, alive = black, died = red, empty = white.
- Press `s` to save a screenshot to `simulation/game_of_life_oop/screenshots/`.

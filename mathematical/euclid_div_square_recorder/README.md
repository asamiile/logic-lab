# Euclid Div Square Recorder

Records the square subdivision image. The Processing PDF export is translated to PNG output because this project uses py5's built-in image saving path.

```bash
uv run python mathematical/euclid_div_square_recorder/euclid_div_square_recorder.py
```

The sketch saves `screenshots/10_6.png` on startup. Press `s` to save another screenshot.

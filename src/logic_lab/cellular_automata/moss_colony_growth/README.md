# Moss Colony Growth

```bash
uv run python src/logic_lab/cellular_automata/moss_colony_growth/moss_colony_growth.py
```

A cellular growth sketch for moss, lichen, and soft colony textures. Cells spread through nearby growth density and a Perlin moisture field, producing fuzzy organic patches instead of rigid geometric grids.

## Controls

- Press `1`, `2`, or `3` for soft moss, lichen islands, and forest floor presets.
- Press `r` to restart the current preset.
- Press `s` to save a screenshot to `cellular_automata/moss_colony_growth/screenshots/`.

Good for moss, lichen, ground cover, fungal colonies, botanical surface texture, and organic spreading fields.

# Audio Wave Scope

```bash
uv run python src/logic_lab/research/audio_wave_scope/audio_wave_scope.py
```

Generated audio-like signals are displayed as an oscilloscope trace, a Lissajous phase view, and a compact spectrum. It avoids live audio dependencies while preserving the visual language of waveform analysis.

## Controls

- Press Space to switch signal models.
- Press `s` to save a screenshot to `research/audio_wave_scope/screenshots/`.

Good for oscilloscope visuals, synthetic audio interfaces, signal studies, music graphics, and waveform-driven compositions.

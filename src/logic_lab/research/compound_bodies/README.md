# Compound Bodies

```bash
uv run python src/logic_lab/research/compound_bodies/compound_bodies.py
```

- Lollipop-shaped compound bodies (stick + candy circle) spawn and fall onto two static platforms.
- Each lollipop is a single physics body with two shapes attached.
- Shapes that fall off the screen are removed from the physics world.
- Press `s` to save a screenshot to `research/compound_bodies/screenshots/`.

# Exercise 6.13 Force Directed Graph

```bash
uv run python research/force_directed_graph_6_13/force_directed_graph.py
```

- Generates 8 random clusters connected by springs.
- `c`: toggle graph connections, `p`: toggle particles, `n`: generate a new graph.
- Press `s` to save a screenshot to `research/force_directed_graph_6_13/screenshots/`.

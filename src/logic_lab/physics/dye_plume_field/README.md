# Dye Plume Field

```bash
uv run python src/logic_lab/physics/dye_plume_field/dye_plume_field.py
```

Dye plume field combines moving dye particles with a low-resolution density field. Particles rise with buoyancy, curl through noise-driven flow, and leave a translucent concentration field behind them.

## Controls

- Hold the mouse to inject dye.
- Press Space to pause or resume automatic emission.
- Press `p` to switch dye palette.
- Press `r` to reset the plume.
- Press `s` to save a screenshot to `physics/dye_plume_field/screenshots/`.

Good for ink dropped in water, colored smoke plumes, underwater dye trails, soft fluid motion, and translucent atmospheric clouds.

# Seismic Wave Field

```bash
uv run python src/logic_lab/physics/seismic_wave_field/seismic_wave_field.py
```

Earthquake events emit fast P-wave rings and slower S-wave rings across layered terrain lines. The waves fade as their radius expands, creating a map-like view of propagation timing.

## Controls

- Click to add a seismic event.
- Press `s` to save a screenshot to `physics/seismic_wave_field/screenshots/`.

Good for impact maps, sonar pulses, earthquake diagrams, radar sweeps, and expanding shockwave compositions.

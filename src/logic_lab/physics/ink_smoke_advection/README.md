# Ink Smoke Advection

```bash
uv run python src/logic_lab/physics/ink_smoke_advection/ink_smoke_advection.py
```

Ink smoke advection simulates watercolor-like pigment clouds spreading through space. Translucent particles are carried by a curl-noise flow field, slowly diffuse outward, and accumulate on a lightly textured paper background.

## Controls

- Hold the mouse to inject ink at the pointer.
- Press Space to pause or resume automatic emission.
- Press `p` to switch pigment palette.
- Press `v` to toggle flow vectors.
- Press `r` to reset the paper and ink.
- Press `s` to save a screenshot to `physics/ink_smoke_advection/screenshots/`.

Good for watercolor ink clouds, smoke plumes, dye spreading through water, soft translucent color fields, and atmospheric backgrounds.

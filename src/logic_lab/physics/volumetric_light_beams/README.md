# Volumetric Light Beams

```bash
uv run python src/logic_lab/physics/volumetric_light_beams/volumetric_light_beams.py
```

Volumetric light beams render light as visible cones passing through a fog particle field. Particles brighten when they fall inside a beam, creating scattering, haze, and soft depth cues.

## Controls

- Press `r` to regenerate the fog particles.
- Press `s` to save a screenshot to `physics/volumetric_light_beams/screenshots/`.

Good for god rays, forest light, stage lighting, fog, theatrical beams, and atmospheric depth.

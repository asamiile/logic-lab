# Particle Force Fields

```bash
uv run python src/logic_lab/physics/particle_force_fields/particle_force_fields.py
```

Particles respond to radial attraction, vortex force, and drag. The field can
be used as a reusable visual layer for magnetic lines, orbital swirls, sparks,
or abstract flow-driven motion.

- Hold the mouse to move the field center.
- Press `s` to save a screenshot.

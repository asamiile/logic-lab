# Wave Equation Grid

```bash
uv run python src/logic_lab/physics/wave_equation_grid/wave_equation_grid.py
```

A discrete two-dimensional wave equation propagates height through a grid. Each cell uses neighboring heights and the previous frame to move energy outward with light damping.

## Controls

- Click or drag to inject wave energy.
- Press `s` to save a screenshot to `physics/wave_equation_grid/screenshots/`.

Good for water membranes, shock propagation, interactive ripples, impact fields, and numerical wave studies.

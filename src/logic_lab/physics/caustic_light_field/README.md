# Caustics Approximation

```bash
uv run python src/logic_lab/physics/caustic_light_field/caustic_light_field.py
```

Caustic light fields approximate the bright curves formed when waves or glass concentrate light. This sketch samples vertical rays through an animated water surface, bends each ray by the local wave slope, and accumulates the ray landings into a soft density field.

## Controls

- Press `v` to toggle sample rays.
- Press `s` to save a screenshot to `physics/caustic_light_field/screenshots/`.

Good for underwater shimmer, glass focus patterns, magical light pools, caustics, and luminous density textures.

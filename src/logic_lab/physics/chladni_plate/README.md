# Chladni Plate

```bash
uv run python src/logic_lab/physics/chladni_plate/chladni_plate.py
```

Particles drift toward the nodal lines of a vibrating rectangular plate. The field combines two sinusoidal standing-wave modes, and each particle follows the gradient toward lower vibration amplitude.

## Controls

- Press `a` or `b` to change the two plate modes.
- Press `r` to reset the sand particles.
- Press `s` to save a screenshot to `physics/chladni_plate/screenshots/`.

Good for resonance patterns, sand-on-plate visuals, ornamental nodal geometry, acoustic diagrams, and cymatic textures.

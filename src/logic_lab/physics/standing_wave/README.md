# Standing Wave

```bash
uv run python src/logic_lab/physics/standing_wave/standing_wave.py
```

A fixed string is drawn as a standing wave mode. The mode number controls the count of nodes and antinodes, while the animated phase shows the oscillation between positive and negative displacement.

## Controls

- Press `+` or `-` to change the mode.
- Press `v` to toggle faint lower modes.
- Press `s` to save a screenshot to `physics/standing_wave/screenshots/`.

Good for string vibration, resonance diagrams, musical harmonics, nodal studies, and clean waveform composition.

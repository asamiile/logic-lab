# Refraction Through Glass

```bash
uv run python src/logic_lab/physics/refraction_prism/refraction_prism.py
```

This sketch bends colored light rays through a triangular glass prism using Snell's law. Slightly different refractive indices split the beam into colored paths, making the refraction angle visible.

## Controls

- Hold and move the mouse vertically to change the incoming beam angle.
- Press `s` to save a screenshot to `physics/refraction_prism/screenshots/`.

Good for prisms, lenses, transparent objects, bent backgrounds, optics diagrams, and spectral light.

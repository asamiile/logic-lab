# Wave Packet

```bash
uv run python src/logic_lab/physics/wave_packet/wave_packet.py
```

A sine carrier is shaped by a Gaussian envelope to form a traveling wave packet. The carrier phase moves inside the packet while the envelope moves at a separate group velocity.

## Controls

- Press `d` to toggle packet spreading.
- Press `s` to save a screenshot to `physics/wave_packet/screenshots/`.

Good for quantum-like pulses, sound bursts, signal envelopes, group velocity demonstrations, and elegant animated linework.

# Ripple Field

```bash
uv run python src/logic_lab/physics/ripple_field/ripple_field.py
```

Circular wavefronts spread from point sources and are sampled as a height field. Each ripple has a moving radius, sinusoidal phase, and exponential decay, so overlapping rings create soft constructive and destructive interference.

## Controls

- Click to add a ripple source.
- Press `s` to save a screenshot to `physics/ripple_field/screenshots/`.

Good for rain-on-water surfaces, impact rings, sonar pings, portal pulses, and layered aquatic textures.

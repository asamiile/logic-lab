# Stable Fluids 2D

```bash
uv run python src/logic_lab/physics/stable_fluids_2d/stable_fluids_2d.py
```

A compact grid-based fluid approximation with density advection, diffusion, and
velocity feedback. It complements particle fluids with a field-oriented solver.

- Hold and drag the mouse to inject dye and velocity.
- Press `s` to save a screenshot.

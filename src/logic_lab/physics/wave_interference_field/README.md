# Wave Interference Field

```bash
uv run python src/logic_lab/physics/wave_interference_field/wave_interference_field.py
```

Coherent radial waves are emitted from several sources and summed at every sample point. Bright bands appear where phases align, while dark nodal bands appear where waves cancel.

## Controls

- Click to add a source.
- Press `v` to toggle source markers.
- Press `r` to reset sources.
- Press `s` to save a screenshot to `physics/wave_interference_field/screenshots/`.

Good for water-wave interference, moire fields, acoustic nodes, energy maps, and scientific texture studies.

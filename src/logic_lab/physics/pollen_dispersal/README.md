# Pollen Dispersal

```bash
uv run python src/logic_lab/physics/pollen_dispersal/pollen_dispersal.py
```

Pollen dispersal simulates lightweight botanical particles carried by wind, swirl noise, and weak gravity. Presets cover pollen clouds, dandelion drift, and seed bursts.

## Controls

- Press `1`, `2`, or `3` for pollen cloud, dandelion drift, and seed burst presets.
- Press `r` to restart the current preset.
- Press `s` to save a screenshot to `physics/pollen_dispersal/screenshots/`.

Good for pollen, seeds, dandelion drift, windborne particles, botanical atmosphere, dispersal, and soft natural motion.

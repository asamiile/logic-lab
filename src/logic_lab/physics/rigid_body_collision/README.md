# Rigid Body Collision

```bash
uv run python src/logic_lab/physics/rigid_body_collision/rigid_body_collision.py
```

Elastic circle bodies collide with each other and with the canvas bounds using
impulse resolution. It is a compact stand-in for a small rigid-body engine:
mass, restitution, friction, gravity, penetration correction, and angular
orientation are all visible.

- Click to add a new body.
- Press `s` to save a screenshot.

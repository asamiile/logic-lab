# Fourier Wave Synthesis

```bash
uv run python src/logic_lab/mathematical/fourier_wave_synthesis/fourier_wave_synthesis.py
```

Canonical waveforms are reconstructed by adding sine harmonics. The sketch shows the combined waveform and faint individual harmonic components, making additive synthesis visible as animated linework.

## Controls

- Press Space to switch square, sawtooth, and triangle waves.
- Press `+` or `-` to change the number of harmonics.
- Press `s` to save a screenshot to `mathematical/fourier_wave_synthesis/screenshots/`.

Good for signal processing visuals, synthesizer studies, waveform design, harmonic education, and rhythmic line animation.

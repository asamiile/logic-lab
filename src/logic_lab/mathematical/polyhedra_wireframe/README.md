# Polyhedra Wireframe

Rotating 3D platonic solids (tetrahedron, cube, octahedron) projected to 2D with perspective.

```bash
uv run python src/logic_lab/mathematical/polyhedra_wireframe/polyhedra_wireframe.py
```

Press `s` to save a screenshot. Press `n` to cycle between polyhedra.

Continuous rotation reveals 3D structure through 2D wireframe projection.

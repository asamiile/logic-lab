# Petal Morphogenesis

```bash
uv run python src/logic_lab/mathematical/petal_morphogenesis/petal_morphogenesis.py
```

Petal morphogenesis generates layered floral forms from polar petal profiles. It controls petal count, radial layering, width, notch depth, and twist to create flower heads whose petals feel biological rather than purely geometric.

## Controls

- Press `1`, `2`, or `3` for camellia, dahlia, and water lily presets.
- Press `s` to save a screenshot to `mathematical/petal_morphogenesis/screenshots/`.

Good for petals, flower heads, radial botanical forms, blooming compositions, floral ornaments, and organic symmetry.

# Reflection Billiards

```bash
uv run python src/logic_lab/mathematical/reflection_billiards/reflection_billiards.py
```

Reflection billiards traces a narrow light ray through a field of mirror segments. Each collision uses the mirror tangent and normal to reflect the direction vector, producing a precise optical path through the scene.

## Controls

- Press Space to pause or resume the beam angle animation.
- Press `r` to reset the launch angle.
- Press `s` to save a screenshot to `mathematical/reflection_billiards/screenshots/`.

Good for kaleidoscopic lines, reflection mazes, geometric path studies, mirror puzzles, and angular composition.

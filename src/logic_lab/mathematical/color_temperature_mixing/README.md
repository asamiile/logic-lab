# Color Temperature Field

```bash
uv run python src/logic_lab/mathematical/color_temperature_mixing/color_temperature_mixing.py
```

Color temperature mixing converts Kelvin values into RGB light colors, then blends several sources additively across a field. Warm, neutral, and cool lights overlap to create a spatial lighting palette.

## Controls

- Press `s` to save a screenshot to `mathematical/color_temperature_mixing/screenshots/`.

Good for neon palettes, sunsets, mixed studio lighting, multi-source illumination, gradient fields, and color planning.

# Bark Growth Texture

```bash
uv run python src/logic_lab/mathematical/bark_growth_texture/bark_growth_texture.py
```

Bark growth texture creates vertical ridges, cracks, scars, and mossy patches from layered noise fields. It provides a botanical material surface rather than a plant silhouette.

## Controls

- Press `1`, `2`, or `3` for old bark, birch scars, and mossy trunk presets.
- Press `s` to save a screenshot to `mathematical/bark_growth_texture/screenshots/`.

Good for bark, tree trunks, wood texture, cracks, mossy surfaces, botanical materials, and organic background texture.

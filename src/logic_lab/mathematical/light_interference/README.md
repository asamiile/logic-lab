# Diffraction and Interference Pattern

```bash
uv run python src/logic_lab/mathematical/light_interference/light_interference.py
```

Light interference sums coherent circular waves from multiple sources. Bright bands appear where wave phases reinforce each other, while dark bands appear where phases cancel.

## Controls

- Press `2`, `3`, `4`, or `5` to change the number of coherent sources.
- Press `t` to toggle threshold bands.
- Press `s` to save a screenshot to `mathematical/light_interference/screenshots/`.

Good for rainbow bands, hologram-like fields, ripple interference, wave optics, diffraction studies, and precise stripe patterns.

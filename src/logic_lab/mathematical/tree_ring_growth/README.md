# Tree Ring Growth

```bash
uv run python src/logic_lab/mathematical/tree_ring_growth/tree_ring_growth.py
```

Tree ring growth draws noisy concentric annual rings with seasonal density variation, eccentric growth, and radial growth marks. It is useful for tree cross sections, wood grain studies, cut stumps, and visualizing botanical time.

## Controls

- Press `1`, `2`, or `3` for oak cross section, old cedar, and burled wood presets.
- Press `m` to toggle radial growth marks.
- Press `s` to save a screenshot to `mathematical/tree_ring_growth/screenshots/`.

Good for annual rings, wood cross sections, tree time, cut stumps, botanical history, and organic circular texture.

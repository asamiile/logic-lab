# Phyllotaxis Flower

```bash
uv run python src/logic_lab/mathematical/phyllotaxis_flower/phyllotaxis_flower.py
```

Phyllotaxis places seeds, florets, or leaves around the golden angle. The result produces sunflower heads, daisy-like florets, pinecone scales, and succulent rosettes with strong botanical structure.

## Controls

- Press `1`, `2`, or `3` for sunflower seeds, daisy florets, and succulent rosette presets.
- Press `s` to save a screenshot to `mathematical/phyllotaxis_flower/screenshots/`.

Good for flowers, seed heads, rosettes, plant spirals, botanical symmetry, and golden-angle compositions.

# P31M Pattern

Generates a hexagonal lattice pattern with threefold rotation and mirror symmetry, translated from `GenerativeArtWithMath/Ch12_Pattern/P31M`.

```bash
uv run python src/logic_lab/tiling_patterns/p31m_pattern/p31m_pattern.py
```

Click to regenerate the random line and curve motifs. Press `s` to save a screenshot.

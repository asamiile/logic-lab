# IH02 TV08

Combines TV08 vertex deformation with IH02 Bezier edge deformation. The original sliders are translated to keyboard controls.

```bash
uv run python src/logic_lab/tiling_patterns/ih02_tv08/ih02_tv08.py
```

Controls: `a`/`z` adjust `hor`, `d`/`x` adjust `ver`, `r` randomizes colors, `s` saves a screenshot.

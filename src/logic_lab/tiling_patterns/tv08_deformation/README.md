# TV08 Deformation

Interactive hexagonal tile deformation. The original sliders are translated to keyboard controls.

```bash
uv run python src/logic_lab/tiling_patterns/tv08_deformation/tv08_deformation.py
```

Controls: `a`/`z` adjust `hor`, `d`/`x` adjust `ver`, `r` randomizes colors, `s` saves a screenshot.

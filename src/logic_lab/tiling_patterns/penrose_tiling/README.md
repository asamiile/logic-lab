# Penrose Tiling

Generates Penrose-style aperiodic tilings from golden triangle subdivisions, translated from `GenerativeArtWithMath/Ch14_Aperiodic/PenroseTiling`.

```bash
uv run python src/logic_lab/tiling_patterns/penrose_tiling/penrose_tiling.py
```

Controls:

- Click: advance one subdivision
- `1`: pentagon/star mode
- `2`: rhomb mode
- `3`: kite/dart mode
- `r`: reset current mode
- `s`: save screenshot

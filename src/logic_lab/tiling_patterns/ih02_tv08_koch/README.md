# IH02 TV08 Koch

Combines TV08 vertex deformation with Koch-subdivided hexagonal tile edges.

```bash
uv run python src/logic_lab/tiling_patterns/ih02_tv08_koch/ih02_tv08_koch.py
```

Controls: `a`/`z` adjust `hor`, `d`/`x` adjust `ver`, `c` increases subdivision, `r` randomizes colors, `s` saves a screenshot.

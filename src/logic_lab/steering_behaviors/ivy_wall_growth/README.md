# Ivy Wall Growth

```bash
uv run python src/logic_lab/steering_behaviors/ivy_wall_growth/ivy_wall_growth.py
```

A steering-based plant growth sketch where tendrils climb a wall, avoid rectangular obstacles, branch, and grow leaves along their paths. It is useful for ivy, creeping vines, wall plants, and botanical line systems that react to architecture.

## Controls

- Press `1`, `2`, or `3` for wall ivy, window climber, and wild creeper presets.
- Press `r` to restart the current preset.
- Press `s` to save a screenshot to `steering_behaviors/ivy_wall_growth/screenshots/`.

Good for ivy, climbing plants, vines around windows, architecture-overgrown compositions, and obstacle-aware botanical growth.

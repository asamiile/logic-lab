# Curl Noise Vines

```bash
uv run python src/logic_lab/steering_behaviors/curl_noise_vines/curl_noise_vines.py
```

Curl noise vines advect many growing line agents through a rotational noise field. The trails read as climbing vines, wind-blown grass, root hairs, or fine botanical strokes.

## Controls

- Press `1`, `2`, or `3` for climbing vines, wind grass, and root hairs presets.
- Press `r` to restart the current preset.
- Press `s` to save a screenshot to `steering_behaviors/curl_noise_vines/screenshots/`.

Good for vines, grass, roots, flowing plant fibers, botanical trails, wind motion, and organic line textures.

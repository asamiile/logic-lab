# Flocking With Binning

```bash
uv run python src/logic_lab/steering_behaviors/flocking_with_binning/flocking_with_binning.py
```

- Boids flock using separation, alignment, and cohesion.
- Spatial binning limits neighborhood checks to nearby grid cells.
- Drag the mouse to add boids.
- Press `s` to save a screenshot to `steering_behaviors/flocking_with_binning/screenshots/`.

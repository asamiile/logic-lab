# Gray-Scott Reaction-Diffusion System

Simulation of chemical interactions between two substances creating self-organizing patterns. The system models the concentration dynamics of two chemicals under diffusion and reaction, producing distinct visual patterns.

```bash
uv run python src/logic_lab/reaction_diffusion/gray_scott/gray_scott.py
```

Press `1/2/3` to select preset patterns (spots/stripes/waves), `↑/↓` to adjust feed rate, `Space` to pause, or `s` to save a screenshot.

The interplay between feed rate (F) and kill rate (k) produces distinct patterns: spots (isolated circles), stripes (wave structures), or waves (traveling fronts).

# Branch Objects Animation

```bash
uv run python fractals/branch_objects_animation/branch_objects_animation.py
```

- Animated recursive tree growth using array-based branching.
- Each branch grows for a set time, then splits into two child branches at different angles.
- When tree reaches max capacity (1024 branches), leaves are added at branch endpoints.
- Press `s` to save a screenshot to `fractals/branch_objects_animation/screenshots/`.

# Differential Growth

```bash
uv run python src/logic_lab/fractals/differential_growth/differential_growth.py
```

Differential growth evolves a closed outline by attracting neighboring points, repelling nearby non-neighbors, and inserting points when edges stretch too far. It creates ruffled leaf edges, coral-like outlines, and soft membrane forms.

## Controls

- Press `1`, `2`, or `3` for ruffled leaf, coral edge, and cell membrane presets.
- Press `r` to restart the current preset.
- Press `s` to save a screenshot to `fractals/differential_growth/screenshots/`.

Good for leaves, coral, frilled petals, biological membranes, organic borders, and non-geometric growth silhouettes.

# Leaf Venation

```bash
uv run python src/logic_lab/fractals/leaf_venation/leaf_venation.py
```

Leaf venation grows branch networks inside a botanical leaf mask. It uses the space colonization principle, but constrains attractor points to leaf silhouettes so the result reads as midrib, side veins, and fine vascular growth.

## Controls

- Press `1`, `2`, or `3` for broad leaf, willow leaf, and heart leaf presets.
- Press `a` to show or hide attractor points.
- Press `r` to restart the current preset.
- Press `s` to save a screenshot to `fractals/leaf_venation/screenshots/`.

Good for leaves, plant anatomy, veins, vascular networks, botanical drawings, and organic branching inside a shaped boundary.
